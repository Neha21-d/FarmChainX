# Farm Chain X - Agricultural Traceability System

A complete full-stack agricultural traceability system that enables end-to-end tracking of crops from farm to consumer using modern web technologies, Spring Boot backend, and Python-based AI services.

**Farm Chain X** - Bringing transparency to the agricultural supply chain through modern web technology! üå±

## üéØ How to Use

### 1. Login & Role Selection
- Open the application in your browser
- Select your role (Farmer, Distributor, Retailer, Consumer, or Admin)
- Enter your name (optional)
- Click "Login as [Role]"

### 2. Farmer Workflow
- **Upload Crop**: Add crop details, quantity, location, and image
- **Generate QR Code**: Automatic QR code generation for traceability
- **Quality Scoring**: AI-powered quality assessment
- **View My Crops**: See all uploaded crops and their status

### 3. Distributor Workflow
- **View Inventory**: See crops available for distribution
- **Forward Crops**: Select and forward crops to retailers
- **Track Distribution**: Monitor crop movement through supply chain

### 4. Retailer Workflow
- **Receive Crops**: View crops forwarded from distributors
- **Mark Available**: Set pricing and mark crops as available for sale
- **Manage Inventory**: Track retail inventory status

### 5. Consumer Workflow
- **Scan QR Codes**: Use the QR scanner to trace food products
- **View Journey**: See complete crop journey from farm to table
- **Scan History**: View all previous scans and traceability data

### 6. Admin Workflow
- **System Overview**: Monitor entire system with analytics
- **Manage Users**: View and manage all system users
- **Track Crops**: Monitor all crops and their status across the supply chain
